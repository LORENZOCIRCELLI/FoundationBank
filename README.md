# FoundationBank Project

Este é um projeto de estudo de um sistema bancário simples desenvolvido em Java com Spring Boot.  
O projeto inclui funcionalidades básicas de gerenciamento de usuários, contas e transações.

Atualmente, a estruturação do sistema está planejada para possuir:

- Entidades: User, Account, Transaction
- Serviços para criação, atualização e listagem de usuários e contas
- DTOs e Responses para transferência de dados entre API e Service
- Banco de dados (configuração local para testes)
